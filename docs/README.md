# Documentation

This repo now separates generated guidance docs from code at the root.

## Structure

- `docs/knowledge/`: canonical long-form docs produced during architecture and integration work.
- `docs/phases/`: phased reading and execution order to imply intent and progress.
- `docs/vector/`: vector-index metadata and ingestion notes for agent retrieval.

## Start Points

- `docs/phases/PHASED_EXECUTION.md` for phase-by-phase execution.
- `docs/knowledge/START_HERE.md` for role-based navigation.
