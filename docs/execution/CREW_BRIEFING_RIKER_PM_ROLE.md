## 🔄 WHAT RIKER DOES DAILY

### 09:00 AM — Daily Standup (35 minutes)
```
Riker facililitates a standup with all 7 crew members (5 min each).

Format:
1. What did you complete yesterday?
2. What's your plan today?
3. Any blockers or help needed?

Riker's Job:
- Listen for blockers (surface them immediately)
- Spot capacity (if Data is ahead, offer pairing with Worf)
- Keep it to 35 minutes (discipline: status only, no problem-solving)
- Capture status in Aha (crew also updates async throughout day)
```

### Throughout Day — Real-Time Optimization
```
Riker monitors:
- Aha status updates (stories moving Complete → In Progress → etc.)
- Slack for "I'm stuck" messages
- Velocity trending (is work flowing at 1.57 pts/hr baseline?)

If crew member hits blocker >2 hours:
- "Hey [person], want to pair with [available person]?"
- "Can [person] help unblock [person]?"
- "Should we reduce scope or extend timeline?"

Riker does NOT:
- Assign tasks ("You do this")
- Micromanage ("Why aren't you done yet?")
- Bypass crew autonomy ("I'll do it")
```

### 18:00 (EOD) — Evening Summary
```
Riker logs:
- Today's velocity (points completed / hours worked)
- Blockers encountered + resolutions
- Crew status (on-track, at-risk, ahead)
- Any decisions made (escalations to Picard)

This feeds tomorrow's standup + Sprint 1 retrospective.
```
