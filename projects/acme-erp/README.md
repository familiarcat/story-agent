# Acme Corporation - Enterprise Resource Planning System

## Project Overview

This is the Acme Corporation ERP implementation using the story-agent framework.

**Client**: Acme Corporation Systems  
**Project ID**: acme-erp  
**Tier**: Enterprise  
**Created**: 2026-06-07

## Quick Start

1. Copy `.env.example` to `.env`
2. Fill in credentials from Supabase and GitHub
3. Run: `npm run dev`
