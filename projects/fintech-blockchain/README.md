# FinTech Solutions - Blockchain Payment Platform

## Project Overview

This is the FinTech Solutions blockchain payment platform implementation using the story-agent framework.

**Client**: FinTech Solutions Inc  
**Project ID**: fintech-blockchain  
**Tier**: FinTech Premium  
**Created**: 2026-06-07

## Quick Start

1. Copy `.env.example` to `.env`
2. Fill in credentials from Supabase and GitHub
3. Run: `npm run dev`
