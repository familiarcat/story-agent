# Healthcare Analytics - Patient Data Intelligence Platform

## Project Overview

This is the Healthcare Analytics patient data intelligence implementation using the story-agent framework.

**Client**: Healthcare Analytics Inc  
**Project ID**: healthtech-analytics  
**Tier**: Healthcare Plus  
**Created**: 2026-06-07

## Quick Start

1. Copy `.env.example` to `.env`
2. Fill in credentials from Supabase and GitHub
3. Run: `npm run dev`

## Features

- HIPAA compliance built-in
- Real-time data analytics
- Patient data security
- Automated migrations
- Health monitoring
- Crew personal memories
- Multi-tenant support with strict isolation
